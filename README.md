imageloader
===========

Use this function to get the iamge width and height

### usage:
    _imgReady(src, ready[, load[, error]]);

### example:
    _imgReady("http://stwww.surfingmagazine.com/wp-content/blogs.dir/1/files/2012/06/girl-in-just-a-shirt-4.jpg", function() {
        alert(this.width);
    });
